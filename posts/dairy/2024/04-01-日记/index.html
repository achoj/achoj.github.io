def convert_markdown_to_html(md_content):
    """Convert Markdown content to HTML."""
    md = markdown.Markdown(extensions=['md_in_html', 'meta', 'toc', 'tables', 'fenced_code'])
    html_content = md.convert(md_content)
    post_date = md.Meta.get('post-date', '')
    edit_date = md.Meta.get('edit-date', '')
    title = md.Meta.get('title', '')
    tags = md.Meta.get('tags', [])
    toc_content = md.toc
    return title, html_content, post_date, edit_date, tags, toc_content

HTML（HyperText Markup Language）是一种用于创建网页的标记语言。它通过使用标签来描述网页的结构，包括文本、图像、超链接等元素的排列方式和显示方式。HTML是构建万维网的基础，它允许开发者创建具有丰富内容和交互性的网页。

HTML最初由蒂姆·伯纳斯-李（Tim Berners-Lee）于1991年创建，目的是让物理学家在不同位置之间共享文档。随着时间的推移，HTML不断发展演变，现在的HTML5版本支持更多的功能和特性，如音频、视频、本地存储等。

HTML基础

HTML文档由一系列的元素（element）组成，每个元素用标签（tag）来表示。标签通常是成对出现的，包含一个开始标签和一个结束标签，内容位于这两个标签之间。

HTML常用标签
标签 描述
<html> 定义HTML文档
<head> 定义文档的头部，包括元数据和外部链接
<body> 定义文档的主体内容
<h1> - <h6> 定义标题，从最重要的到最不重要的标题
<p> 定义段落
<a> 定义超链接
  • HTML标签是大小写不敏感的，但是建议使用小写字母，以保持一致性。
  • HTML文档应该以<!DOCTYPE html>声明开头，这样浏览器可以按照最新的HTML标准来解析文档。
  • 注释可以通过<!-- 这是注释 -->标签来添加到HTML文档中。

HTML文本格式化

HTML提供了各种标签来格式化文本，使其更具可读性和吸引力。例如，可以使用<strong>标签来加粗文本，使用<em>标签来斜体显示文本。

HTML (HyperText Markup Language) is a markup language used to create web pages. It describes the structure of a web page using tags to arrange and present elements such as text, images, hyperlinks, etc. HTML serves as the foundation for building the World Wide Web, allowing developers to create web pages with rich content and interactivity.

HTML was initially created by Tim Berners-Lee in 1991 with the goal of allowing physicists to share documents across different locations. Over time, HTML has evolved, and the current version HTML5 supports more features and functionalities such as audio, video, local storage, etc.

HTML Basics

An HTML document consists of a series of elements, each represented by tags. Tags are typically paired, containing an opening tag and a closing tag, with content between them.

Common HTML Tags
Tag Description
<html> Defines the HTML document
<head> Defines the document's head, including metadata and external links
<body> Defines the document's main content
<h1> - <h6> Defines headings, from most important to least important
<p> Defines a paragraph
<a> Defines a hyperlink
  • HTML tags are case-insensitive, but it is recommended to use lowercase for consistency.
  • An HTML document should start with a <!DOCTYPE html> declaration to ensure browsers parse the document according to the latest HTML standards.
  • Comments can be added to HTML documents using the <!-- This is a comment --> tag.

Formatting Text in HTML

HTML provides various tags for formatting text to make it more readable and appealing. For example, you can use the <strong> tag to bold text, and the <em> tag to italicize text.
